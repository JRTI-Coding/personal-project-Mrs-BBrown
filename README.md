# Unity Project Template

⚠️ **Important Rules (Read First)**

- Do NOT rename folders
- Do NOT move files
- Open this project ONLY from **Unity Hub**
- This project is already connected to GitHub

---

## How to Start
1. Accept the GitHub Classroom assignment
2. Open **GitHub Desktop** and clone the repository
3. Open **Unity Hub**
4. Click **Add / Locate**
5. Select this project folder (the one with `Assets`)

---

## Daily Workflow
1. Open project from **Unity Hub**
2. Work in Unity
3. Open **GitHub Desktop**
4. Write a clear commit message
5. **Commit** → **Push**

---

## If Something Breaks
- Do NOT delete random files
- Ask for help before starting over

Your work is graded using your **GitHub commit history**.
